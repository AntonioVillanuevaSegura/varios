Une analyse d'une manière possible de lire et d'enregistrer des données à partir d'un fichier de configuration.
Chaque ligne de configuration est de type {key}:{value} , 
les commentaires /* */ // ou les espaces ne sont pas pris en compte, 
le fichier externe est lu et enregistré dans un tableau {array} de dictionnaires {key:value }
Où le dictionnaire a autant d'entrées que de lignes de configuration a le fichier source externe
